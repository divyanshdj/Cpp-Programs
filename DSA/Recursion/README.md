# 🔁 Recursion

Recursion is a process where a function calls itself until a base condition is met.

## 🔸 Types
- Tail & Head recursion
- Linear, Tree, Indirect recursion

## 🔸 Common Examples
- Factorial, Fibonacci
- Power(x, n)
- Sum of digits
- Print N to 1 / 1 to N

> Helps build problem-solving mindset through repeated subproblem reduction.
